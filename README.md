rabbitmq-grpc

## About
  This repository is a test of communication between systems using RabbitMQ and gRPC with nestjs framework.

## How to start

### Start consumer and producer API
  ```js
    cd consumer/
    yarn
    yarn start:dev
  ```
  
  ```js
    cd producer/
    yarn
    yarn start:dev    
  ```
  
### Call producer API
  Use the consumer API Rest to request the producer API.
